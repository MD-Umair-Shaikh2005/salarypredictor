Salary Predictor ML Project
📌 Description
This project predicts salary based on input features using Machine Learning.

📂 Files
dataset.csv → dataset used
model.ipynb → Jupyter Notebook with training and prediction
⚙️ Requirements
Python
pandas
scikit-learn
numpy
🚀 How to Run
Open model.ipynb in Jupyter Notebook
Run all cells
Use the model for prediction
🤝 Contribution
Frontend developers can build a UI and connect it to this model.
